# 人体姿态估计  

人体姿态估计，Human Pose Estimation， HPE  
1. 图像或视频中，人体关节（关键点）的定位问题。  
2. 所有关节姿势组成的空间中搜索特定姿势。  

# 难点

1. 关节小而灵活  
2. 衣服遮挡  

# 已有方法

1. 基于部件组合的传统方法  
2. DeepPose  
3. 卷积 + 滑动窗口 + 热图  
4. 卷积摆位机  
5. 误差反馈  
6. 堆叠式沙漏网络  
7. 高分辨率网络

# 估计指标

1. PCP：正确部件的百分比  
2. PCK：正确关键点的百分比  
3. OKS：对象关键点相似度
