1. Q-Learning

  1. 使用一个π\piπ与环境互动，生成labelled data
  2. 根据labelled data学到Qπ(s,a)Q^\pi(s, a)Qπ(s,a)
  3. 根据Qπ(s,a)Q^\pi(s, a)Qπ(s,a)找到更好的π′\pi'π′
  4. 用π′\pi'π′代替π\piπ，goto step 1

1.1. 怎么评价哪个π\piπ更好？

答： Vπ′(s)≥Vπ(s)V^{\pi'}(s) \ge V^\pi(s)Vπ′(s)≥Vπ(s) for all state s，则认为π′\pi'π′比π\piπ好。

1.2. 怎么根据Qπ(s,a)Q^\pi(s, a)Qπ(s,a)找到π′\pi'π′？

答：
π′(s)=argmaxaQπ(s,a) \pi'(s) = \arg\max_a Q^\pi(s, a) π​′​​(s)=arg​a​max​​Q​π​​(s,a)

continuous a不适用。

1.3. 怎么根据labelled data学到Q？

答： Target Network

注意：
因为左边的QπQ^\piQπ和右边的QπQ^\piQπ是同一个模型，当左边的QπQ^\piQπ因为迭代而更新时，右边的QπQ^\piQπ也会改变。
左边QπQ^\piQπ更新 --> 右边QπQ^\piQπ改变 --> 右边的输出改变 --> 左边的目标改变。
如果一个NN的目标一直在改变，这个NN就会很难train。
解决方法：右边的QπQ^\piQπ先fix住，训练左边的QπQ^\piQπ。左边的QπQ^\piQπ更新一定次数后把它参数同步到右边的QπQ^\piQπ。

1.5. Replay Buffer

这是一个用于存互动经验的buffer，每一项的格式为st,at,rt,st+1s_t, a_t, r_t, s_{t+1}st​,at​,rt​,st+1​
每一轮迭代都可以产生N个data，但buffer可以开很大，里面可以存很多轮data。
因此step 2用于生成Q的labelled data不止是都个π\piπ的互动data。

优点：

  1. 充分利用data，减少与环境互动的时间
  2. batch里面的data越diverse越好。这些方法便data更diverse。

1.6. Q-Learning的过程

  1. 初始化两个NN，分别是QQQ和Q^\hat QQ^​，且Q^=Q\hat Q = QQ^​=Q
  2. given sts_tst​，take ata_tat​ based on Q(epsilon greedy)
  3. 得到rtr_trt​和st+1s_{t+1}st+1​，把(st,at,rt,st+1)(s_t, a_t, r_t, s_{t+1})(st​,at​,rt​,st+1​)存入buffer
  4. 从buffer sample一个batch的data

  5. 计算target：
    y=ri+maxaQ^(si+1,a) y = r_i + \max_a \hat Q(s_{i+1}, a) y=r​i​​+​a​max​​​Q​^​​(s​i+1​​,a)

  6. 更新Q的参数，使Q(si,ai)Q(s_i, a_i)Q(si​,ai​)接近y（回归问题），goto step 4

  7. 令Q^=Q\hat Q = QQ^​=Q

1.7. Q-Learning存在的问题：Q value的估测值高于实际值。

为什么Q value的估测值会特别高？

Q(st,at)=rt+maxQ(st+1,at) Q(s_t, a_t) = r_t + \max Q(s_{t+1}, a_t) Q(s​t​​,a​t​​)=r​t​​+maxQ(s​t+1​​,a​t​​)

Q是一个NN，可能会对某个action高估，某些action低估。
而max倾向于选择被高估的action。

解决方法：Double DQN
